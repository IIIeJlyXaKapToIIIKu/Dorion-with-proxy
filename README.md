# Dorion with proxy

Набор скриптов для запуска [Dorion](https://github.com/SpikeHD/Dorion) через HTTP-прокси с логином и паролем.

Dorion сам умеет принимать параметр `--proxy`, но WebView2 может всё равно показывать окно авторизации прокси. Эти скрипты обходят проблему через локальный прокси-bridge: Dorion подключается к `127.0.0.1:18080` без авторизации, а bridge уже добавляет `Proxy-Authorization` при подключении к внешнему прокси.

## Что нужно

- Windows.
- Установленный Dorion, рядом с которым лежит `Dorion.exe`.
- Установленный Python 3. Проверить можно командой:

```powershell
python --version
```

## Установка

Скопируйте эти файлы в папку, где находится `Dorion.exe`:

```text
Dorion.bat
Dorion-with-proxy.vbs
Dorion-with-proxy.ps1
proxy_bridge.py
Setup-proxy.bat
Setup-proxy.ps1
```

## Первоначальная настройка

Запустите:

```text
Setup-proxy.bat
```

В открывшемся окне введите прокси в одном из форматов:

```text
http://login:password@host:port
```

или:

```text
host:port:login:password
```

После нажатия `Save` рядом со скриптами появится файл `proxy-config.json`. В нём хранится выбранный прокси, включая логин и пароль. Не публикуйте этот файл и не отправляйте его другим людям.

## Запуск Dorion

После настройки запускайте:

```text
Dorion.bat
```

`Dorion.bat` запускает скрытый PowerShell-менеджер, который:

1. Поднимает локальный прокси на `127.0.0.1:18080`.
2. Запускает `Dorion.exe --proxy http://127.0.0.1:18080`.
3. Ждёт закрытия Dorion.
4. Автоматически завершает локальный proxy bridge.

Если `proxy-config.json` ещё не создан, при запуске `Dorion.bat` окно настройки откроется автоматически.

## Смена прокси

Чтобы изменить прокси, снова запустите:

```text
Setup-proxy.bat
```

Старое значение будет подставлено в поле ввода. Замените его и нажмите `Save`.

## Проверка, что локальный прокси остановился

Пока Dorion открыт:

```powershell
Test-NetConnection 127.0.0.1 -Port 18080
```

Ожидаемо:

```text
TcpTestSucceeded : True
```

После закрытия Dorion повторите команду. Ожидаемо:

```text
TcpTestSucceeded : False
```

## Безопасность

`proxy-config.json` содержит credentials открытым текстом. Файл добавлен в `.gitignore`, но всё равно проверяйте, что он не попал в архив, репозиторий или сообщение перед отправкой кому-либо.
